🚀 PRÓXIMOS PASSOS:

1️⃣  INSTALAR DEPENDÊNCIAS
    npm install

2️⃣  CONFIGURAR VARIÁVEIS
    cp .env.example .env
    # Edite .env e coloque:
    # - DISCORD_TOKEN (de discord.com/developers)
    # - DISCORD_CLIENT_ID

3️⃣  (OPCIONAL) FIREBASE
    # Opção A: arquivo JSON
    # Coloque firebase-credentials.json na raiz
    
    # Opção B: variáveis de ambiente
    # Edite .env com FIREBASE_*

4️⃣  INICIAR BOT
    npm start
    # Ou com reload automático:
    npm run dev

5️⃣  CRIAR SEUS COMANDOS
    cp TEMPLATE_COMANDO.js commands/MinhaCategoria/novo.js
    # Editar e salvar - carrega automaticamente!
